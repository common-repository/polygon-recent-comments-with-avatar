Polygon Recent Comments With Avatar: Recent comments with avatar support, including Gravatar, date, username, user link, and scrollbar.

== Description ==

Display recent comments in the sidebar with user avatar/Gravatar support, styles, information, and an active scrollbar for handling numerous comments.
<a href="https://wordpress.org/plugins/polygon-recent-comments-with-avatar/faq/" target="_blank">FAQ</a>
<a href="https://wikipoly.com/en/poly-comments/" target="_blank">Support and request additional features as needed</a>

== Installation ==

1. Upload the `polygon-recent-comments-with-avatar` folder to the `/wp-content/plugins/` directory.
2. Activate the Polygon Recent Comments With Avatar plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin by going to the `Widgets` section in your WordPress dashboard and selecting "Polygon Recent Comments".

== Frequently Asked Questions ==

How to hide the scrollbar when your comments are displayed?

You can set the "Comment display frame height (in pixels)" to a large value (e.g., 1000) or uncheck "Show scroll?"
